# Projeto Teclado CAA
Projeto desenvolvido na cadeira de Introdução à Computação no primeiro período da graduação de Sistemas de Informação na CESAR School.

## Descrição
O desafio proposto foi de reciclar de forma criativa um componente de computador (hardware ou periférico), transformando-o em outro objeto.

O projeto consiste em um teclado físico modificado para utilizar 21 teclas ilustradas com imagens descritivas, utilizando a Comunicação Aumentativa e Alternativa (CAA), utilizada para auxiliar pessoas com dificuldades severas de fala ou linguagem.
